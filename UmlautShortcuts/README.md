### Umlaut Shortcuts

Adds shortcuts to write umlauts [ä,ö,ü,ß,Ä,Ö,Ü] on a US keyboard layout or any other layout.

<details>
  <summary><b>Shortcuts</b></summary>

* `Win + A` - ä
* `Win + Shift + A` - Ä


* `Win + O` - ö
* `Win + Shift + O` - Ö


* `Win + U` - ü
* `Win + Shift + U` - Ü


* `Win + S` - ß

</details>
